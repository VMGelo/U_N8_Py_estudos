# API - Application Programing Interface

1) Conjunto de rotinas e padrçoes de programação para acesso a um aplicativo
2) Desenvolvimento em **back-end** que permite compartilhar informações entre aplicações Web e outras aplicações Nativas. É uma camada de comunicação facilmente compreendida, usando linguagem JSON. (Interoperabilidade).
3) Sempre vai existir um Client side e um Server Side, onde a API faz a intermediação dessas informações/chamadas. O cliente pergunta, o Servidor responde.
4) Recursos - São elementos de informação que através de um identificador global podem ser manipulados. 
    - Identificação do recurso:
        - URL - Uniform Resource Locator
        - URI - Uniform Resource Identifier
        - URN - Uniform Resource Name
